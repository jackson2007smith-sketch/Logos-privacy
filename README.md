# Logos Privacy Policy

This folder contains the public GitHub Pages privacy policy for Logos.

## Before publishing

The public support email currently used by the policy is:

```text
Logosreadingapp@gmail.com
```

## Publish with GitHub Pages

1. Add `index.html` to the root of the privacy-policy GitHub repository.
2. Commit and push it to the `main` branch.
3. Open the repository's **Settings > Pages**.
4. Select **Deploy from a branch**.
5. Choose `main` and `/(root)`, then save.

The public address will usually be:

```text
https://YOUR-GITHUB-USERNAME.github.io/YOUR-REPOSITORY/
```

Use that address for the Privacy Policy URL in App Store Connect.
